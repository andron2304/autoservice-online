# AutoService Online

## Про проєкт
AutoService Online — це вебзастосунок для автосервісу, створений для автоматизації взаємодії між власниками автомобілів та працівниками СТО. Система дозволяє клієнтам швидко знаходити вільні вікна для обслуговування, а працівникам — зручно планувати робочий час та вести облік виконаних робіт.

## Основний функціонал

### Для клієнта:
* Реєстрація та створення профілю.
* Додавання автомобілів до власного гаража (марка, модель, рік, VIN-код).
* Перегляд каталогу послуг та актуальних цін.
* Бронювання часу в розкладі конкретного майстра.
* Перегляд історії записів та відстеження статусів ремонту.

### Для адміністратора та працівників:
* Управління розкладом: перегляд усіх заявок на день або тиждень.
* Керування статусами записів (Очікує підтвердження, В роботі, Готово, Скасовано).
* Ведення довідника послуг (редагування списку робіт та цін).

## Технологічний стек
* Мова програмування: Python
* Веб-фреймворк: Django
* База даних: PostgreSQL (pgAdmin 4)
* Контроль версій: Git, GitHub
* Управління проєктом: GitHub Projects (Kanban)

## Інструкція із розгортання

1. Клонуйте репозиторій:
    ```bash
    git clone [https://github.com/andron2304/autoservice-online.git](https://github.com/andron2304/autoservice-online.git)
    ```

2. Створіть та активуйте віртуальне середовище:
    ```bash
    python -m venv venv
    source venv/bin/activate  # для Linux/macOS
    venv\Scripts\activate     # для Windows
    ```

3. Встановіть необхідні залежності:
    ```bash
    pip install -r requirements.txt
    ```

4. Виконайте міграції бази даних:
    ```bash
    python manage.py migrate
    ```

5. Запустіть сервер розробки:
    ```bash
    python manage.py runserver
    ```
